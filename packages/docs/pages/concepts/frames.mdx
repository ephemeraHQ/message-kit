# Frames

FrameKit allows you to generate frames from your backend agent.

## Payments

```tsx
context.requestPayment(payment: Payment)
```

## Custom Frames

```tsx
context.sendCustomFrame(frame: string)
```

## Receipts

```tsx
context.sendReceipt(receipt: Receipt)
```

## Profiles

```tsx
context.sendProfile(profile: Profile)
```

## Groups

```tsx
context.sendGroup(group: Group)
```
