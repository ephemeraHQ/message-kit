# AI Skills

AI Skills system framework enables agents to understand and process skills through multiple intelligence layers, creating more natural and powerful conversational experiences.

![](/img/skills.png)

## Overview

MessageKit steps to create fully functional AI agents that perform complex tasks:

- [Reasoning](/concepts/skills/reasoning): Agent can handle complex conversations and perform autonomously.
- [Scenarios](/concepts/skills/scenarios): Train your agent for reliable responses.
- [Prompting](/concepts/skills/prompting): System prompt to teach available skills to the agent.
- [Processing](/concepts/skills/processing): Handle multiple skills in a single conversation.
- [Parsing](/concepts/skills/parsing): Interpret user intent and translate it into a defined skill.
- [Definition](/concepts/skills/definition): Define skills, parameters and handlers.
