# Examples

All of these examples are open source and made with MessageKit.

## Live examples

- [Wordle Bot](https://converse.xyz/dm/wordlebot.eth): Play wordle in a group chat. [Repo](https://github.com/ephemerahq/wordle-bot)
- [Brian Bot](https://converse.xyz/dm/brian-agent.eth): Agent for all defi built with brian.
- [Base Agent](https://converse.xyz/dm/baseagent.converse.xyz): Base agent for all swaps, mints and payments, all in base. [Repo](https://github.com/humanagent/base-agent)
- [ENS Agent](https://converse.xyz/dm/ensagent.converse.xyz): All about ens domains. [Repo](https://github.com/humanagent/ens-agent)

## Quickstarts

Other open source quickstarts of agents built with MessageKit or Deployment help.

- [Frensday](https://github.com/ephemeraHQ/frensday): Frensday is a agent that managed frensday poaps, updates, and more.
- [Next.js](https://github.com/humanagent/message-kit-next-quickstart): Deploy in a vercel app.
- [Broadcast bot](https://github.com/humanagent/broadcast-bot): Broadcast messages to subscribers through XMTP..
- [gm bot](https://github.com/xmtp/gm-bot): A simple bot that sends a gm message to a user.

## Open Frames Quickstarts

These are quickstarts for different use cases inside of our Docs

- [Tx Pay](https://github.com/humanagent/txpay): A frame for triggering EVM **mobile** transactions. Also shows receipts.
- [Base Tx Frame](https://github.com/ephemeraHQ/base-tx-frame): A dev-oriented frame designed for creating transactions on Base.
- [OnchainKit Quickstart](https://github.com/ephemeraHQ/open-frames-starter-onchainkit/): Onchain Kit quickstart that integrates XMTP.
- [Frog Quickstart](https://github.com/ephemeraHQ/open-frames-starter-frog): Frog XMTP quickstart
- [Frames.js Quickstart](https://github.com/ephemeraHQ/open-frames-starter-framesjs/): Frames.js quickstart that integrates XMTP.
- [Converse Deeplinks](https://replit.com/@FabrizioGuespe/Converse-Deeplink#index.js): Converse Deeplink for creating your own custom links.
