# Message Kit

A set of tools for AI developers to build on top of [XMTP](https://xmtp.org)

> Try XMTP using [xmtp.chat](https://xmtp.chat)

- [`xmtp`](./xmtp/library): An node js wrapper for AI agents around the [node-sdk](https://github.com/xmtp/xmtp-js/tree/main/sdks/node-sdk)
- [`resolver`](./xmtp/resolver): A suite of tools for resolving identites to EVM addresses
