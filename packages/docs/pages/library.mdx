## Getting Started

To use any of these skills in your project, you can install MessageKit using your preferred package manager:

:::code-group

```bash [bun]
bun create message-kit
```

```bash [yarn]
yarn create message-kit
```

```bash [npx]
npx create-message-kit
```

```bash [npm]
npm init message-kit
```

:::
