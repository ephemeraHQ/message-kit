## Introduction

Here's how to easily deploy this app on Replit by importing an existing GitHub repository:

- Sign up at [Replit](https://replit.com/).
- Create a new Replit and select "Import from GitHub".
- Connect your GitHub account and select the repository you want to import.
- Set your environment variables in the Replit Secrets tab.
- Run the project to start your application.
- Register an [ENS domain](https://ens.domains/) and share your app!

## Deployment

1. **Sign Up and Setup**: Create an account at [Replit](https://replit.com/) and start a new Replit.

2. **Import Repository**: Select "Import from GitHub" and connect your GitHub account.

3. **Environment Variables**: Set your environment variables in the Replit Secrets tab.
