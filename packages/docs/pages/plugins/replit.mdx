## Introduction

Here's how to easily deploy this app on Replit:

- Sign up at [Replit](https://replit.com/).
- Fork the [MessageKit Quickstart Template](https://replit.com/@raihankhan-rk/MessageKit-Quickstart-Template).
- Set your environment variables in the Replit Secrets tab.
- Run the project to start your application.
- Register an [ENS domain](https://ens.domains/) and share your app!

## Deployment

1. **Sign Up and Setup**: Create an account at [Replit](https://replit.com/) and fork the Quickstart Template.

![](/img/replit/1.png)

2. **Environment Variables**: Set your environment variables in the Replit Secrets tab.

![](/img/replit/2.png)

- **Run the Project**: Click the "Run" button to start your application.
