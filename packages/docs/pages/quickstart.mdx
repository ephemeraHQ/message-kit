### CLI

This tool simplifies the creation and configuration of new agents.

- [Replit Quickstart Template](https://replit.com/@raihankhan-rk/MessageKit-Quickstart-Template): The fastest way to get started.

- Or choose one of the templates from the CLI to get started.

:::code-group

```bash [bun]
bun create message-kit
```

```bash [yarn]
yarn create message-kit
```

```bash [npx]
npx create-message-kit
```

:::

![](/img/cli/3.png)

### Variables

Set up these variables in your app

```bash [cmd]
KEY= # the private key of the agent wallet (generated by the CLI)
TEST_ENCRYPTION_KEY= # the private key of the test wallet (generated by the CLI)
OPENAI_API_KEY= # the API key for OpenAI
```

### Run the bot

Follow the steps below to run the bot

:::code-group

```bash [bun]
# Install the dependencies
bun install
# run the app
bun dev
```

```bash [yarn]
# Install the dependencies
yarn install
# run the app
yarn dev
```

```bash [npm]
# Install the dependencies
npm install
# run the app
npm run dev
# or
npm run start
```

:::
