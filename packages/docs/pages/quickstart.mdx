### Templates

### Variables

Set up these variables in your app

```bash [cmd]
KEY= # the private key of the agent wallet
OPENAI_API_KEY= # the API key for OpenAI
```

### Run the bot

Follow the steps below to run the bot

:::code-group

```bash [bun]
# Install the dependencies
bun install
# run the app
bun dev
```

```bash [yarn]
# Install the dependencies
yarn install
# run the app
yarn dev
```

```bash [npm]
# Install the dependencies
npm install
# run the app
npm run dev
# or
npm run start
```

:::
