### Replit or CLI

- [Replit Quickstart Template](https://replit.com/@raihankhan-rk/MessageKit-Quickstart-Template): The fastest way to get started.

- Or choose one of the templates from the CLI to get started.

:::code-group

```bash [bun]

```

```bash [yarn]
yarn create message-kit
```

```bash [npx]
npx create-message-kit
```

![](/img/cli/2.png)

### Variables

Set up these variables in your app

```bash [cmd]
KEY= # the private key of the agent wallet (generated by the CLI)
TEST_ENCRYPTION_KEY= # the private key of the test wallet (generated by the CLI)
OPENAI_API_KEY= # the API key for OpenAI
```

### Run

Follow the steps below to run the bot

:::code-group

```bash [bun]
# Install the dependencies
bun install
# run the app
bun dev
```

```bash [yarn]
# Install the dependencies
yarn install
# run the app
yarn dev
```

```bash [npm]
# Install the dependencies
npm install
# run the app
npm run dev
# or
npm run start
```

:::

## Compatible apps

These wallets are built on top of XMTP and are EVM compatible, menaing they work with any wallet address or ens domain.

- [**Converse**](https://converse.xyz)
- [**Coinbase Wallet**](https://www.coinbase.com/en-ar/wallet/downloads)
- [**Warpcast**](https://warpcast.com/)
- [**Family**](https://family.co/)

## Manual Installation

You can also install the package manually using your preferred package manager.

:::code-group

```bash [bun]
bun i @xmtp/message-kit
```

```bash [npm]
npm i @xmtp/message-kit
```

```bash [yarn]
yarn add @xmtp/message-kit
```

:::
