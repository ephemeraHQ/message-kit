# Examples

Here are some ready to go examples of agents built with MessageKit. Documented and ready to use.

## Templates

These are templates for different use cases inside of our Docs

- [`gm`](./templates/gm): Simple gm bot example.
- [`group`](./templates/group): Group chat bot example with skills and examples.
- [`agent`](./templates/agent): ENS agent for registering and checking domains

### Quickstarts

Other open source quickstarts of agents built with MessageKit or Deployment help.

- [Frensday](https://github.com/ephemeraHQ/frensday): Frensday is a bot that managed frensday poaps, updates, and more.
- [Event invite bot](https://github.com/fabriguespe/invite-bot): Invite bot for Events linked to a Notion db.
- [Simple AI bot](https://github.com/fabriguespe/gpt-example): Simple GPT example.
- [Subscription bot](https://github.com/fabriguespe/subscription-bot): Subscription bot example for a newsletter.
- [Next js](https://github.com/fabriguespe/message-kit-next-quickstart): Deploy in a vercel app.
- [Gated group](https://github.com/ephemerahq/gated-group): Group chat with a bot that can add members to a group.
- [Faucet bot](https://github.com/fabriguespe/faucet-bot): Faucet bot for testnet.

### Frames quickstarts

These are quickstarts for different use cases inside of our Docs

- [Base Tx Frame](/directory/basetxframe): A dev-oriented frame designed for creating transactions on Base.
- [Tx Receipt Frame](/directory/txreceipt): A frame designed for displaying transaction receipts.
- [OnchainKit Quickstart](https://github.com/ephemeraHQ/open-frames-starter-onchainkit/): Onchain Kit quickstart that integrates XMTP.
- [Frog Quickstart](https://github.com/ephemeraHQ/open-frames-starter-frog): Frog XMTP quickstart
- [Frames.js Quickstart](https://github.com/ephemeraHQ/open-frames-starter-framesjs/): Frames.js quickstart that integrates XMTP.
