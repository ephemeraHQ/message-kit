# ENS Agent

This is a simple yet powerful agent that will help users interact with ENS.

## Skills

- [Check](/skills/library/check): Check if a domain is available
- [Cool](/skills/library/cool): Show cool alternatives
- [Register](/skills/library/register): Register a domain
- [Reset](/skills/library/reset): Reset a domain
- [Renew](/skills/library/renew): Renew a domain
- [Info](/skills/library/info): Get info about a domain
- [Pay](/skills/library/pay): Pay for a domain
