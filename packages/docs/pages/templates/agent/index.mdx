```
agent/

```

## Run the app

Follow the steps below to run the app

::::steps

### Setup

```bash [cmd]
# Clone the repo
git clone https://github.com/ephemeraHQ/message-kit
# Go to the templates/agent folder
cd templates/agent
# Install the dependencies
yarn install
# Run the app
yarn dev
```

### Variables

Set up these variables in your app

