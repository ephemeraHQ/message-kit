# Coinbase Agent

This is a simple yet powerful agent that will help users interact with Coinbase.

## Structure

```
coinbase-agent/
├── src/
│   ├── plugins/
│       ├── learnweb3.ts
│       ├── redis.ts
│   └── skills/
│       ├── drip.ts
│       ├── mint.ts
│       ├── pay.ts
│       ├── swap.ts
│   ├── index.ts
│   ├── prompt.ts
├── package.json
├── tsconfig.json
└── .env
```

### Skills

- [Drip](/skills/drip): Drip a coin
- [Mint](/skills/mint): Mint a coin
- [Pay](/skills/pay): Pay for a coin
- [Swap](/skills/swap): Swap a coin

## Plugins

- [LearnWeb3](/plugins/learnweb3): Interact with LearnWeb3
- [Redis](/plugins/redis): Interact with Redis
