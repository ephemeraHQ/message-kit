# ENS Agent

This is a simple yet powerful agent that will help users interact with ENS.

## Structure

```
ens/
├── src/
│   └── skills/
│       ├── check.ts
│       ├── cool.ts
│       ├── info.ts
│       ├── register.ts
│       ├── renew.ts
│       ├── reset.ts
│       ├── pay.ts
│   ├── index.ts
│   ├── prompt.ts
├── package.json
├── tsconfig.json
└── .env
```

### Skills

- [Check](/skills/check): Check if a domain is available
- [Cool](/skills/cool): Show cool alternatives
- [Register](/skills/register): Register a domain
- [Reset](/skills/reset): Reset a domain
- [Renew](/skills/renew): Renew a domain
- [Info](/skills/info): Get info about a domain
- [Pay](/skills/pay): Pay for a domain
