

## Run the app

Follow the steps below to run the app

::::steps

### Setup

```bash [cmd]
# Clone the repo
git clone https://github.com/ephemeraHQ/message-kit
# Go to the templates/gated folder
cd templates/gated
# Install the dependencies
yarn install
# Run the app
yarn dev
```
