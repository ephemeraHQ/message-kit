# Tipping

In this example you can tip using 3 primitives:

```bash [cmd]
/tip @bo @alix 15
```

## Skill declaration

How the skill is declared in the `skills.ts` file:

```tsx [src/skills.ts]
export const skills: SkillGroup[] = [
  {
    name: "Group bot",
    tag: "@bot",
    description: "Group agent for tipping payments, games and more.",
    skills: [
      /* Other skills */
      {
        skill: "/tip [usernames] [amount] [token]",
        description: "Tip users in a specified token.",
        handler: tipping,
        params: {
          username: {
            plural: true,
            default: "",
            type: "username",
          },
          amount: {
            default: 10,
            type: "number",
          },
        },
      },
      /* Other skills */
    ],
  },
];
```
