# Hackathon Store

Get all the hackathon needs from the general store.

## Structure

```
hackathon-store/
├── src/
│   ├── plugins/
│       ├── notion.ts
│   ├── index.ts
│   ├── prompt.ts
├── package.json
├── tsconfig.json

```
