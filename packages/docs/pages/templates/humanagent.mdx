# Human Agent

Create wallets for users and interact with them.

## Structure

```
humanagent/
├── src/
│   └── skills/
│       ├── waas.ts
│   └── vibes/
│       ├── degen.ts
│   ├── index.ts
│   └── prompt.ts
├── package.json
├── tsconfig.json
└── .env
```

- [Waas](/skills/waas): Agent Wallet as a Service
- [Degen](/vibes/degen): Define the personality of your agent
