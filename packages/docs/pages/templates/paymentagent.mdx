# Payment Agent

Create wallets for users and interact with them.

> Payment Agent uses [CDP plugin](/plugins/cdp) to interact with Coinbase.

## Structure

```
paymentagent/
├── src/

│       ├── degen.ts
│   ├── index.ts
│   └── prompt.ts
├── package.json
├── tsconfig.json
└── .env
```
