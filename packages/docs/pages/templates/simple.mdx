# Simple Agent

Here’s a step-by-step guide to get you started.

::::steps

### CLI

This tool simplifies the creation and configuration of new apps.

:::code-group

```bash [bun]
bun create message-kit
```

```bash [yarn]
yarn create message-kit
```

```bash [npx]
npx create-message-kit
```

```bash [npm]
npm init message-kit
```

:::

_Refer to [structure](/concepts/structure) of the app so see the folder is created._

### Variables

Set up these variables in your app

```bash [cmd]
KEY= # the private key of the agent wallet
OPENAI_API_KEY= # the API key for OpenAI
```

### Run the bot

Follow the steps below to run the bot

:::code-group

```bash [bun]
# Install the dependencies
bun install
# run the app
bun dev
```

```bash [yarn]
# Install the dependencies
yarn install
# run the app
yarn dev
```

```bash [npm]
# Install the dependencies
npm install
# run the app
npm run dev
# or
npm run start
```
