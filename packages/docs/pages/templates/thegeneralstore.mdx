# The General Store

Get all the hackathon needs from the general store.

## Structure

```
thegeneralstore/
├── src/
│   ├── plugins/
│       ├── notion.ts
│       ├── lowdb.ts
│       ├── redis.ts
│       ├── learnweb3.ts
│   └── skills/
│       ├── faucet.ts
│       ├── notion.ts
│       ├── poap.ts
│   ├── index.ts
│   ├── prompt.ts
├── package.json
├── tsconfig.json
└── .env
```

### Skills

- [Faucet](/skills/faucet): Get tokens from the faucet
- [Mint](/skills/mint): Mint a coin
- [Pay](/skills/pay): Pay for a coin
- [Swap](/skills/swap): Swap a coin

### Plugins

- [Notion](/plugins/notion): Interact with Notion
- [LearnWeb3](/plugins/learnweb3): Interact with LearnWeb3
- [LowDB](/plugins/lowdb): Interact with LowDB
- [Redis](/plugins/redis): Interact with Redis
