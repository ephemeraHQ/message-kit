## Setup

Follow these steps to set up and run the project:

1. **Navigate to the project directory:**

```sh
cd ./mk-langhchain
```

2. **Set up your environment variables:**

```sh
KEY= # the private key of the wallet
OPENAI_API_KEY= # sk-proj-...
```

3. **Install dependencies:**

```sh
yarn@10.9.0 install
```

4. **Run the project:**

```sh
yarn@10.9.0 dev
```

5. Enjoy!
